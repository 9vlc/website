> Start of this website
This all started as a spontaneous urge to learn HTML.
After looking at other people's personal websites for so long - nishi.boats and msx.horse, for
example, I decided it would be fun to try to make my own.

Blatantly inspired by msx.horse's design, I decided to go with a static navbar on the left and
the rest of the website on the right. Not going to bore you with the CSS horror stories I went
through while trying to make this in your regular modern HTML5+CSS, I had a complete mental
breakdown over the modern web industry and rewrote everything in strict W3C HTML 4.01 to not go
any more insane over this. Now, CSS here is only being used for some eyecandy like fonts on
modern web browsers, everything else is being handled by regular HTML.


> About the domain name
I initially wanted this to be a joke website.

Making a joke website, you need to find a hilarious domain name, of course. I decided to go to
{no advertising} on a search for available peculiar domain names, during which many untaken
domains have surfaced, some notable ones being:
  • doohickey.diy,
  • iommu.horse,
  • furries.dating,
  • spymalware.com, (and)
  • greg.exposed
After some (not very lengthy) consideration, paidbsd.org was chosen.

If anyone from the FreeBSD people are reading this, I don't want to infringe on anything, I
just find this domain name funny.
