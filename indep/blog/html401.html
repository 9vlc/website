Quick update: I reworked this website to be more strictly compatible with HTML 4.01 and have a bit more of backwards compatibility with older browsers.

Current state of browser support:
  • Retrozilla seems to render most of the website correctly
  • Retro Ladybird doesn't render pages with framesets
  • Dillo fails to render framesets but the website is useable
  • WINE iexplore fails to open local files, HTTP error permanent redirect on Cloudflare pages
  • Links works perfectly (relative to its minimalism)
  • Lynx does not support frames, fails with permanent redirect
  • w3m does not support frames, fails with permanent redirect
